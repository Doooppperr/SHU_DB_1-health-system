## 8. 问题修复记录（补充）

1. 现象  
登录/注册时提示失败，但刷新后已登录。
2. 根因  
前端成功后跳转目标写为不存在的路由名 `home`，异常被误捕获后显示“失败”。
3. 修复  
登录页与注册页成功跳转统一改为 `institutions`。
4. 验证  
`npm run build` 成功；手工验证登录/注册后不再误报失败。
